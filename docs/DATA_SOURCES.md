## üîç How to Use These Links

### Finding Data Sources

1. **Start at Main Portal**: https://dbh.hkdir.no/tall-og-statistikk/

2. **Navigate to Student Statistics**: 
   - Statistikk-meny ‚Üí Studenter ‚Üí Statistikk-side 4.1

3. **Drill Down by Hierarchy**:
   - Select institution type (Universiteter / Statlige vitenskapelige h√∏yskoler)
   - Select institution
   - Select faculty/department (optional)
   - View courses

4. **Copy URL**: The URL in your browser contains all filter parameters

### Extracting Parameters

The URLs contain encoded parameters:
- `insttype`: Institution type (11 = Universiteter, 12 = Statlige h√∏yskoler)
- `instkode`: Institution code
- `arstall`: Year (optional - omit for all years)
- `nivakode`: Education level codes
- `hier`: Hierarchy structure

### Using URLs to Find New Institutions

1. Visit NSD portal
2. Navigate to desired institution
3. Copy URL from browser
4. Extract institution code from URL parameters
5. Use code in discovery scripts

## üìä URL Parameter Reference

### Common Parameters

| Parameter | Description | Example Values |
|-----------|-------------|----------------|
| `visningId` | View ID | `205` |
| `insttype` | Institution type | `11` (Universiteter), `12` (Statlige h√∏yskoler) |
| `instkode` | Institution code | `1110` (UiO), `1120` (UiB), `1150` (NTNU), `1240` (NHH) |
| `arstall` | Year | `2025`, `2024`, etc. (omit for all years) |
| `nivakode` | Education level | `b3`, `b4`, `hk`, `yu`, `ar`, `ln`, `m2`, `me`, `mx`, `hn`, `m5`, `pr` |
| `hier` | Hierarchy path | Encoded path like `insttype%219%21instkode%219%21emnekode` |
| `sti` | Display path | Human-readable path (URL encoded) |

### Education Level Codes (nivakode)

- `b3`: Bachelor 3-year
- `b4`: Bachelor 4-year
- `hk`: H√∏gskolekandidat
- `yu`: Yrkesutdanning
- `ar`: Artium
- `ln`: L√¶rerutdanning
- `m2`: Master 2-year
- `me`: Master (other)
- `mx`: Master (extended)
- `hn`: H√∏gskoleingeni√∏r
- `m5`: Master 5-year
- `pr`: PhD

## üí° Tips for Finding New Sources

1. **Start at Main Portal**: Always start at https://dbh.hkdir.no/tall-og-statistikk/

2. **Follow the Hierarchy**: Navigate through the drill-down menus

3. **Copy URLs**: The URL contains all filter parameters needed

4. **Check Institution Codes**: Verify code matches in URL (instkode parameter)

5. **Test in Browser**: Open the URL to verify it shows the data you want

6. **Use "All Years"**: Remove `arstall` parameter or set to all years for complete data

## üìù Notes

- URLs are URL-encoded (e.g., `%219%21` = `|!|`)
- Parameters can be modified directly in the URL
- Removing `arstall` parameter gives all years
- The hierarchy path (`hier`) determines drill-down structure
- Institution codes are consistent across URLs
